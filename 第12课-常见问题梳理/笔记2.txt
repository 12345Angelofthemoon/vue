部署：
1.打包
  yarn run build

2.安装nginx
  yum install nginx -y

3.启动nginx
  systemctl start nginx

4.关闭防火墙
  iptables -F

5.清除掉测试文件
  cd /usr/share/nginx/html
  rm ./* -rf

6.文件上传到服务器（winscp）
  

*7.nginx转发
