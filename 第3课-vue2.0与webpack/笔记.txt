loader怎么用

loader配置：
1.单loader
{test: //, use: 'xxx-loader'}

2.多个loader
{test: //, use: ['xxx-loader', 'xxx-loader']}

3.loader+参数
{test: //, use: {
  loader: 'xx-loader',
  options: {
    a: , b:
  }
}}

4.多loader+参数
{test: //, use: [
  {
    loader: 'xxx-loader',
    options: ...
  }, {
    loader: 'xxx2-loader',
    options: xxx
  }
]}

{test: //, use: [
  'xxx-loader', {
    loader: 'xxx2-loader',
    options: xxx
  }
]}

--------------------------------------------------------------------------------

file-loader   读取文件、不做处理、直接输出
url-loader    读取文件、把文件的内容转换成base64，打包到bundle里面

base64：
1.优点——减少请求
2.缺点——略微增加文件体积
  1.3K -> 1.5K
  10%~20%
